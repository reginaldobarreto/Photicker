# Photicker

## About App

This application is very cool, 
because you can take your photos and put emoticons, 
leaving the photos with a relaxed air.

## Skills applied

* Class Field
* Access to Camera by Intent
* Permission Request
* Image manipulation (Zoom in, Zoom out, rotate)
* RecyclerView
* FileProvider
* Thread
